    Steps to Run the program -

    1. Extract the contents of the zip file into a folder.
    2. After changing the working directory to current directory where we unzipped the files, compile the file BinarySearchTree.java in the terminal using 'javac BinarySearchTree.java'
    3. Next run the program by running the command java BinarySearchTree
    4. Enter the numbers to be inserted using positive numbers.
    5. To delete the numbers enter as negative numbers.
    6. The sample input and output as given by professor are shown below.
